# Servicio Web de Registro y Login

## Descripción
Este proyecto implementa un servicio web que permite:
- Registrar usuarios
- Autenticación de usuarios (login)

## Tecnologías
- Node.js
- Express

## Endpoints

### Registro
POST /api/register

### Login
POST /api/login

## Ejecución
1. Instalar dependencias:
npm install

2. Ejecutar servidor:
node server.js
